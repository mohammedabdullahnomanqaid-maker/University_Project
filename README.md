# University_Project
Student Management System 🎓 A C# WinForms app for university records. It consolidates CRUD operations into a single interface for a seamless UX. ​Key Features: 🔹 Unified UI: Manage students (Add/Edit/Delete) in one window. 🔹 Visual Cards: Simplified data presentation. 🔹 Custom Views: Flexible display modes (Icons/Details) for easy navigation.
